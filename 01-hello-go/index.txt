1 - важен импорт main package и функция main, entry файл может называть по-разному
2 - go fmt path - запуск форматировщика го
3 - go get url - скачивание внешнего пакета
4 - go mod tidy - удаляет добавляет либы, которые используются или не используются?
    После go mod tidy в go.mod остаются только используемые библиотеки.

5 - // indirect в go.mod значит что это зависимость зависимости либо зависимость, которую мы не используемые
6 - go get не нужно выполнять при клонировании репы, достаточно go run
