1 - var  - одну переменную
2 - vars - несколько переменных
3 - короткое объявление и инициализация переменных возможна только внутри тела функции
4 - %.{n}f - форматирование числа (n = 1, кол-во цифр после точки)
5 - %T - позволяет вывести тип переменной (fmt.Printf("type is %T", var))
6 - %s - значение
7 - если мы просто объявляем переменную, мы обязаны указать тип (var name string)
8 - переменные имеют значение по умолчанию! 
    a - string = ""
    b - int = 0
    c - bool = false

9 - в go строгая типизация 
    a - нельзя выполнять операции с разными типами данных, например float64 * int, нужно приводить типы

10 - можно перечислять названия переменных var a, b c = 1, 2, 3

11 - const необходимо сразу объявлять и присваивать значение, компилятор не присваивает дефолтные значения

12 - константа может быть untyped и может кастоваться к другому типу 1 + 1.1 (см. пример)
    a - это сделано так, потому что константы не меняют свое значение

13 - можно группировать переменные
    var | const (a = 2 b = ")

14 - можно создать const и не использовать ее, ошибка компиляции не будет в отличие от var
