1 - в коде можно хранить неиспользуемые функции
2 - параметры в функции передаются в момент ее вызова
3 - можно переопределить параметры, присвоить им другие значения
4 - функции могут иметь названия возвращаемого значения, в таком случае они вовращаются автоматически, это называется naked return
5 - нельзя объявлять функции через ключевое слово func внутри других функций, используем функциональные выражения (литерала), присвоенные переменным
6 - _, value - можно пропускать значение
7 - замыкание возникает, когда из одной функции возвращается другая
